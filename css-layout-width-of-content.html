Defining the width of the content area

There are several ways to define the width of a main contet area, which often is done in web layout to accomodate wide screens (or more precisely: wide vieports). This basic approach seems to be a de facto standard in today's web design, although it is certainly not the only solution and therefore may be discussed in general.

For the moment, however, let's just assume, that this strategy is the way we want to go about our sites' overall layout and explore some of the ways of accomplishing the task.
