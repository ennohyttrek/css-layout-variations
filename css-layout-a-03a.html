CSS Layout A 03a

Examples 3a through 3c are to test the difference in several ways of including webfonts in a site. 3a will use the HTML link tag. All fonts are stored locally in site's assets.

The A examples of layouts in this series are based on a very simple structure in HTML: just four first level elements without any surrounding wrappers. Header, nav, main, footer. The inner structures (for example within the main element) may vary, like in this example, which adds a section element to cluster articles.

Section Grid Heading H2

It is actually pretty easy to create a card layout like in this example. Simply use the flexbox and then clever width settings for the flex elements. Please note that controlling a flex layout using the width property for flex elements is an entirely valid method and actually documented as one possible way to control flex layout by W3C.

With this approach one could for example easily turn the normal list view of articles in a standard WordPress setup into an article/post thumbnail grid.
