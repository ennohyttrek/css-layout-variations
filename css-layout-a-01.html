CSS Layout A 01

The A examples of layouts in this series are based on a very simple structure in HTML: just four first level elements without any surrounding wrappers. Header, nav, main, footer. The layout challenge in this example is not very hard: just build a vertical flow layout (like for example a one-page-scroll site would require) and provide a stage area in the center for wider viewports. Almost all horizontal spacing tasks within this solution are handled by padding.
